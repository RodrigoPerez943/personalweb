SPO32

Este proyecto es el más complejo y extenso que he realizado. La idea surgió porque quería unificar varias acciones que realizaba todos los días en diferentes aplicaciones en un único dispositivo. Las funcionalidades implementadas son las siguientes:

  • Reloj digital con la temperatura actual, máxima y mínima del día.
  • Controlador de Spotify que muestra el tiempo restante de la canción en reproducción, permite cambiar de canción, ir a la anterior, pausarla y controlar el volumen.

STYLO-BOX

Este proyecto fue realizado para la asignatura de Sistemas Digitales 2 en la universidad, el proyecto consistía en una parte obligatoria la cual consistía en crear un Jukebox con un STM32 en leguaje C. En la parte final opcional decidí hacer un "stylophone" creando las piezas a partir de unos sensores y con la opción de subir y bajar de octava.

Recibí la insignia de "Proyecto con Mejor diseño Hardware" gracias a él.

AUTO-YT

Quería investigar la monetización de youtube mediante la creación de una canal de automatizado, con la API de Google para youtube y un poco de scraping conseguí generar contenido en muy pocos minutos, sin necesidad de editar ni buscar contenido.

El programa esta diseñado para dos funcionalidades, subir reels de youtube y crear recopilaciones de vídeos obtenidos de instgram. Genera nombres de forma automática para los videos y se pueden incluir descripciones y hastags, o bien usar unos predefinidos.

Basé el código en uno no mantenido que encontré en GitHub, añadiendole infinidad de mejoras y actualizandolo según mis gustos, quedando un programa muy fácil de usar.

Proyectos Para Empresas

TEXT2QU es un software privado que fue diseñado y desarrolado para la automatización de generación de ejercicios de puzzle de programación para cursos telemáticos.

Con él, en cuestion de segundos, es posible generar cientos de preguntas de tipo puzzle orientadas a la programación, de un video de una clase o conferencia grabada de forma automática.

TEXT2Q es un software privado que fue diseñado para la automatización de generación de preguntas para cursos telemáticos para una empresa.

Con él, en cuestion de segundos, es posible generar cientos de preguntas de tipo multiple-choice, verdadero o falso y fill-the-gaps para un vídeo de una clase o conferencia grabada de forma automática.

HACKATHON - EESTech Challenge 2024 - MINSAIT

Tuve el placer de participar en un Hackathon de 3 días en mi universidad en los cuales tuvimos que desarrollar dos modelos de Machine Learning, uno para predecir notas de IMBD según sus propiedades y otro para predecir las notas también pero según el poster de las películas.

Fueron tres días muy intensos, se clasificaron para las finales europeas unos compañeros y la acabaron ganando! Aprendí muchas habilidades nuevas como el procesamiento de imágenes, el funcionamineto de los diferentes modelos de inteligencia artificial y como diseñarlos.
